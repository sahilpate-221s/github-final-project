# Simple Interest Calculator

This project is a simple interest calculator implemented using a Bash script.

## Description
The calculator computes **simple interest** based on the following user inputs:
- Principal amount
- Rate of interest
- Time period (in years)

## Formula Used
Simple Interest = (Principal × Rate × Time) / 100

## Usage
The user is prompted to enter the principal amount, rate of interest, and time period.  
The script then calculates and displays the simple interest.
